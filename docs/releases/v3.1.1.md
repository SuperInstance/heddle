# Heddle v3.1.1

Release date: 2026-06-05

Heddle v3.1.1 is a focused terminal usability release for the control-plane
TUI introduced in v3. It improves how assistant output and prompt entry behave
in everyday terminal sessions.

## Highlights

### Markdown renders cleanly in the TUI

Assistant responses in the terminal now render Markdown instead of showing raw
Markdown syntax for common content.

Why this matters:

- headings, lists, emphasis, links, and inline code are easier to scan
- code blocks are visually separated with a dim terminal background
- code blocks remain copy-friendly instead of adding border characters to the
  copied text
- rendered output is easier to read when the agent returns long analysis or
  documentation-style answers

### Prompt input is less brittle

The TUI prompt composer now behaves better around approval and long-input
workflows.

Why this matters:

- approval shortcut keys such as `r` no longer leak into the prompt input while
  an approval request is active
- long prompts wrap across the terminal width more predictably
- terminal input stays usable for larger prompts and follow-ups

## Upgrade Notes

- No command migration is required from v3.1.0.
- `heddle` and `heddle chat` continue to open the API-backed TUI.
- The active browser UI and mobile path are unchanged.
